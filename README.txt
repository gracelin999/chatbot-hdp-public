Matching Patients to Clinical Trials using a Chatbot
(A consulting project for HDP and Insight)

Note: I've removed some code/info specific to HDP so some functions might look 
funny/incomplete. Contact me if you have questions.


------------------------------------
Run
------------------------------------
Pre-req (if needed)
> unzip drugbank-full-database.xml.zip

Local web GUI with flask:
> python application.py

------------------------------------
Files/directories
------------------------------------
application.py 	: flask user i/f
proc_chat.py	: main Python script
templates/		: used by flask
med-lexicon.txt	: local lexicon; add words as we go
trials.csv		: contains trials used in this project
*.pickle		: supporting pickle files generated by proc_trials.py
proc_trials.py 	: pre-process the trials and cluster them into 
				  decision-tree type of graph used by application.py
